# Huajiuan Allinium

EOSS шилэн пасад, металл пасад, гадна дуулаалга, дагалдах тоноглолын B2B вэб сайт.

## Технологи

- Next.js 15
- React 19
- TypeScript
- Tailwind CSS v4
- next-intl
- Apollo Client
- framer-motion
- lenis

## Суулгах

```bash
pnpm install
```

## Хөгжүүлэлт

```bash
pnpm dev
```

## Build

```bash
pnpm build
```

## Агуулгын менежмент

erxes CMS-тэй холбогдсон. `NEXT_PUBLIC_ERXES_ENDPOINT` болон `NEXT_PUBLIC_ERXES_APP_TOKEN` environment variables шаардлагатай.
